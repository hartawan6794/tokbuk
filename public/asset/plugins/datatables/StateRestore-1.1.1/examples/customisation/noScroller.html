stateRestore example Don't save scroller

This example shows how to use the stateRestore.saveState.scroller initialisation option (documented at stateRestore.saveState) to disable the saving of the table's current scroll position by setting the initialisation option to false. The default value is true meaning that the scroll position will be saved when using scroller.

The Javascript shown below is used to initialise the table shown in this example:

$(document).ready(function() { $('#example').DataTable({ dom: 'Blfrtip', ajax: "../data/2500.txt", deferRender: true, scrollY: 200, scrollCollapse: true, scroller: true, buttons:[ 'createState', { extend: 'savedStates', config: { saveState: { scroller: false } }, } ] }); });
